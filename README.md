# Alliance Tax for SeAT

This plugin automates tax collection for alliances in EVE Online. It scans member ledgers for mining and ratting activity, calculates what they owe based on your configurable rates, and generates invoices automatically.

## Features

*   **Mining Tax**: Tracks moon ore, ice, and standard belt ore.
*   **Ratting Tax**: Tracks bounty prizes (TickRatting) and ESS payouts.
*   **Flexible Rates**: Set global alliance rates or specific rates per corporation.
*   **Automated Invoicing**: Generates consolidated invoices for members (weekly/monthly).
*   **Discord Integration**: Pings members when invoices are ready.
*   **Payment Reconciliation**: Checks alliance wallet journals to automatically mark invoices as paid.
